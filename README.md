# P3

A simple Android application that showcases a basic user interface for a login form. It includes input fields for name, email, and phone number, along with buttons to save or clear the input.

## Features

*   A simple and intuitive user interface.
*   Input fields for name, email, and phone.
*   'Save' button to acknowledge saving the data (with a message).
*   'Clear' button to reset the form.

## How to run

1.  Open the project in Android Studio.
2.  Sync the project with Gradle files.
3.  Run the app on an emulator or a physical device.
